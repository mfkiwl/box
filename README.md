# box

`box` is a text-based visual programming language.

```
 ┌─ƒ(Is Even)────────────────────────────────────────────────────┐
 │                     ┌─/*...*/───────────────────────┐         │
 │                     │ Returns true if input is even │         │

 │ │ number ○┼─┐                                                 │
 │ └─────────┘ │                                                 │
 │             │  ┌─ƒ(Modulo)──┐   ┌─ƒ(Equality)──┐ ┌─Outputs──┐ │
 │             └──┼○          ○┼───┼○            ○┼─┼○         │ │
 │ ┌─2──┐     ┌───┼○           │ ┌─┼○             │ │          │ │
 │ │   ○┼─────┘   └────────────┘ │ └──────────────┘ └──────────┘ │
 │ └────┘               ┌─0──┐   │                               │
 │                      │   ○┼───┘                               │
 │                      └────┘                                   │
 └───────────────────────────────────────────────────────────────┘
```
