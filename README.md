# box

`box` is a text-based visual programming language.

```
                                ┌─[For Loop]─────────┐
                       ┌────────┼►        Loop body ►┼───────────────────────────────────┐
                       │        │                    │     ┌────────────────────┐        │ ┌─ƒ(Format)───┐    ┌─ƒ(Print)──┐
                       │        │                    │     │ "The loop has run" │        └─┼►           ►┼────┼►          │
┌─ƒ(Print N times)──┐  │   ┌────┼○ start             │     │ "{index} times"   ○┼──────────┼○           ○┼────┼○          │
│                  ►┼──┘   │    │                    │     └────────────────────┘        ┌─┼○            │    └───────────┘
│                n ○┼──────┼────┼○ end        index ○┼───────────────────────────────────┘ └─────────────┘
└───────────────────┘      │    │                    │
                           │    │                    │              ┌─ƒ(Print)──┐      ┌─[Return]─┐
       ┌────┐              │    │         Completed ►┼──────────────┼►         ►┼──────┼►         │
       │ 0 ○┼──────────────┘    └────────────────────┘            ┌─┼○          │      └──────────┘
       └────┘                                ┌──────────────────┐ │ └───────────┘
                                             │ "Done looping!" ○┼─┘
                                             └──────────────────┘
```
